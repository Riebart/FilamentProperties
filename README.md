# FilamentPropertyLookup

_Generative AI Disclosure: While I'm a dev, I am horseshit at UI, so I got Perplexity+Claude to poop out at the HTML and CSS for this. All issues with this can be directed into the IDGAF bucket, or a PR._

A little webapp/printable workshop chart for quickly checking on material properties, such as chemical resistance.

Started off based on the data from Prusa on [Prusament](https://prusament.com/chemical-resistance-of-3d-printing-materials/), but quickly spooled because Nylon and TPU weren't there, and same with some common adhesives and chemicals/solvents.

I've added a few filaments and chemicals I found useful or use often.
